图标速览
1. 随机点名计分台保留随机点名。抽到单人时由该玩家答题；抽到全员抢答时可用课堂辅助奖励。
2. 基础得分答对点 +1 或 +2。课堂积分决定排名，军粮用于大富翁经营和战术动作。
3. 职业系统六个职业有被动和 3 回合冷却技能。军师爆发，将军经营，先锋机动，粮官拿资源，策士反领先，参谋控概率。
4. 锦囊装备答题后可能掉锦囊，最多保留 3 张卡。锦囊令加分，藤甲挡租，连策军令下次额外 +1，檄文向第一名收费，沙盘提高锦囊概率。
5. 大富翁经营军粮可建多种建筑并升级；科技能强化经济、建筑、谋略和军事，兵种攻击主要影响军粮、建筑和锦囊牌。
6. 军师远征1 人是单人闯关，2-6 人可合作讨伐。合作闯关会共同计算团队总分，职业轮换答题会触发连携伤害。
职业定位
卧龙军师爆发型。技能自己 +2 并拿锦囊；答对 +2 时有机会额外 +1。适合：冲分 + 锦囊
虎威将军经营型。能免费建造，已有建筑后收租更强；越早铺资产越稳定。适合：建筑 + 收租
白袍先锋机动型。技能得分、拿轻骑兵并推进地图，适合快速积累国力。适合：兵种 + 机动
青囊粮官资源型。更容易掉锦囊，技能能照顾最低分玩家，课堂氛围比较友好。适合：锦囊牌 + 追分
绯羽策士反领先型。可以向第一名收费，遇到大型战鼓事件还有机会拿檄文卡。适合：追击第一名
星河参谋控概率型。能提高锦囊概率并拿沙盘，适合赌传说、神话事件。适合：谋略 + 奇谋
建筑说明
茅庐书院｜3 军粮低价起步，租金 1，适合第一座建筑。升级后收租更稳。便宜铺场
军粮市集｜5 军粮租金 1，每轮有军粮收入。经济路线的核心建筑。稳定赚钱
谋略台｜6 军粮提高锦囊和奇谋事件机会，适合配合青囊粮官、星河参谋和谋略科技。事件增益
虎豹骑营｜7 军粮解锁更好的招兵选择，也让招兵成本更舒服。军事入口
城墙箭楼｜5 军粮被攻击时优先抵挡，保护军粮、锦囊牌和高等级建筑。防守反制
三国军府｜10 军粮高价地标，租金 3 且有收入。适合中后期领先或稳局。高投入高回报
科技与兵种
屯田经济提高答题和建筑收入。前期升 1 级通常最稳。越早越赚
城防工程增加建筑容量，并降低建造和升级压力。适合地产流
谋略奇术提高锦囊和奇谋触发感，爆发不稳定但上限高。适合赌事件
兵法军备解锁更强兵种并降低招兵成本，但需要军粮和兵营支撑。适合压制流
轻骑兵｜3 军粮抢 1 军粮，低成本干扰，不会直接重创课堂积分。轻攻击
弩兵｜4 军粮让对方建筑暂停 1 轮，需要兵营或军事科技支持。克制建筑
谋士｜5 军粮优先打散对方锦囊牌，没有锦囊牌时抢军粮。克制锦囊牌
虎豹骑｜8 军粮强攻击，会削弱建筑并抢军粮，适合后期翻盘。后期爆发
热门发展路线
1. 稳健经营流先建茅庐书院或军粮市集，再升屯田经济和城防工程。特点是收入稳定，适合大多数学生。推荐：虎威将军 / 卧龙军师
2. 锦囊谋略流先攒谋略台，再升谋略奇术，靠锦囊、沙盘和奇谋事件制造反超机会。推荐：青囊粮官 / 星河参谋
3. 兵营压制流先建虎豹骑营，再升兵法军备，购买弩兵或谋士干扰领先者的建筑和锦囊牌。推荐：白袍先锋 / 绯羽策士
平衡性提示
课堂积分优先攻击主要影响军粮、建筑和锦囊牌，不大幅扣课堂积分，避免学生被打到失去参与感。
防滚雪球设计策士、最低分奖励、护盾、城墙箭楼和不能连续攻击同一人的规则，会给落后者留追赶空间。
经济与军事互相牵制只买兵会缺收入，只建市集会怕干扰；通常需要至少一座防守或谋略建筑来补短板。
老师调节建议如果课堂节奏慢，少开兵种，多用全员 +1；如果学生很熟练，再鼓励科技和建筑升级。
