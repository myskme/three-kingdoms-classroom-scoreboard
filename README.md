# 三国课堂军师争霸积分器 iPad 版

这是一个适合课堂投屏和 iPad 使用的静态网页游戏工具。

## 使用方式

直接打开 `index.html` 即可使用。项目不需要安装依赖，也不需要后端服务。

## GitHub Pages 发布

发布源建议选择：

- Branch: `main`
- Folder: `/ (root)`

发布后，GitHub Pages 会直接读取根目录的 `index.html`。

## 文件结构

- `index.html`：主页面和全部游戏逻辑
- `assets/`：三国主题图片、职业头像、图标素材
- `.nojekyll`：让 GitHub Pages 按静态文件原样发布
